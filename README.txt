== Description ==

**Install plugin to previous order of customer from your admin order page**

Easy to use, you only install the plugin.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please contact me on <a href="http://purvik.me">purvik.me</a>.*

== Installation ==

1. Upload `wc-customer-order-status.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
